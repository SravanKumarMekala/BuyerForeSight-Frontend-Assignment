# BuyerForeSight User Dashboard

## Features

User Directory Dashboard built with React.

Functions:

• Fetch users from API  
• Search users  
• Sort users  
• User detail view  
• Loading state  
• Error handling  
• Responsive design  

## Tech Stack

React
React Router
CSS

## Setup

npm install

npm run dev
